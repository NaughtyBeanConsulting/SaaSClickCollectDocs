# About

The About page is a trust and brand-story surface inside the storefront. It helps customers understand the cafe, the brand tone, and why they should order directly instead of treating the site like a generic menu.

## What this flow covers

- Opening the About page from storefront navigation
- Reviewing the brand-story section
- Understanding the call back into the menu

## Open the About page

### 1. Use the **About** item in the storefront navigation

The bottom navigation exposes **About** alongside Menu, Orders, Cart, and Settings.
