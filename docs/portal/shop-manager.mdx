# Shop Manager

Open Shop Manager at `https://clickcollect.coffee/portal/manage/` to run the order queue and manage menu content.

## What is inside Shop Manager

- `Orders`
- `Categories`
- `Extras`
- `Drinks`
- `Settings` for higher-access roles

Staff can work the order queue only. Managers can also manage categories, extras, and drinks. Owners and admins get the full set of tools.

## Card view vs table view

### 1. Open the **Orders** tab and choose the view mode

The live orders area exposes two toggle buttons:

- **Table View** for a denser operational view across many orders
- **Card View** for a more visual, easier-to-scan layout

Use table view when you want to compare many rows quickly. Use card view when you are scanning fewer orders or working on a smaller screen.

## Add categories

### 7. Open the **Categories** tab and add a category

Create the category structure first so drinks have a clean home in the storefront.

Typical categories include `Espresso Drinks`, `Cold Drinks`, or `Seasonal Specials`.

## Add extras

### 8. Open the **Extras** tab and create add-ons

Extras are the modifiers you want to attach to drinks, such as an extra shot or a milk change.

Add the extra name, price, and active state, then save it.

<ScreenshotPlaceholder
  title="Extras tab and add form"
  pathHint="static/img/portal/shop-manager/08-extras.png"
/>

## Add drinks

### 9. Open the **Drinks** tab and add a drink

Work through the drink form with:

- Name
- Description
- Category
- Base price
- Any size adjustments
- Image
- Active state

If the account has no open product capacity left, saving another product can trigger a billing confirmation.

<ScreenshotPlaceholder
  title="Drinks tab and add form"
  pathHint="static/img/portal/shop-manager/09-drinks.png"
/>

## Assign extras to drinks

### 10. Edit the drink and attach extras from the available list

Once the extras already exist, return to the drink form and assign the relevant extras to that drink.

This is the normal sequence:

1. Create the category.
2. Create the extra.
3. Create or edit the drink.
4. Assign extras to the drink.

<ScreenshotPlaceholder
  title="Assign extras to a drink"

## Good practice

- Build categories before you build drinks.
- Add extras once, then reuse them across drinks.
- Use table view for busy shifts and card view for a more visual queue scan.
- Push orders all the way to `Collected` so the active queue stays useful.
