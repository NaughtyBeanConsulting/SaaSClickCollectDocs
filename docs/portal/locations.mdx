# Locations

Open the Locations page at `https://clickcollect.coffee/portal/locations/` to manage your collection branches. Customers choose from these locations during checkout.

## What this page controls

- Branch name and address
- WhatsApp number per branch
- Google Maps link per branch
- Default location selection
- Active or inactive branch state
- Assigned team members per location

## Add a location

### 1. Open **Locations** and review current usage

The page opens with a branch summary and a list of existing locations. The usage card also shows whether another branch could trigger billing.

<ScreenshotPlaceholder
  title="Locations page overview"
  pathHint="static/img/portal/locations/01-locations-overview.png"
/>

### 2. Select **Add New Location**

If the account is still within trial or plan capacity, the button opens the form immediately. If not, the button can be disabled or the save action can later trigger a billing confirmation.

<ScreenshotPlaceholder
  title="Add new location action"
  pathHint="static/img/portal/locations/02-add-location-action.png"
/>

### 3. Complete the location form

The form supports these fields:

- `Name`
- `Address`
- `WhatsApp number`
- `Google Maps URL`
- `Is default`
- `Is active`
- `Assigned team`

Save once the branch details are complete.

<ScreenshotPlaceholder
  title="Location form fields"
  pathHint="static/img/portal/locations/03-location-form.png"
/>

### 4. Confirm billing if the new branch exceeds open capacity

If your account has no open paid location capacity left, saving the new location can show a billing confirmation for the location add-on price.

<ScreenshotPlaceholder
  title="Location billing confirmation"
  pathHint="static/img/portal/locations/04-location-billing-confirm.png"
/>

## Assign team members to a location

### 5. Use the **Assigned team** multi-select while creating or editing the branch

The location form includes a multi-select list of staff members you can attach to the branch.

This is the fastest way to connect a branch to the people who work there.

<ScreenshotPlaceholder
  title="Assigned team multi-select"
  pathHint="static/img/portal/locations/05-assigned-team.png"
/>

### 6. Review the location in the list view

After saving, check the location card or table row for:

- Name and status badge
- WhatsApp number
- Google Maps link
- Assigned team members
- Actions such as **Edit** or **Make Default**

<ScreenshotPlaceholder
  title="Saved location in the list"
  pathHint="static/img/portal/locations/06-saved-location.png"
/>

## Edit, default, or remove a branch

### 7. Use the row actions for branch maintenance

- **Edit** updates the location form.
- **Make Default** promotes that branch to the default pickup option.
- **Delete** removes the location after confirmation.

Only one branch can be the default at a time.

<ScreenshotPlaceholder
  title="Location row actions"
  pathHint="static/img/portal/locations/07-location-actions.png"
/>

## Good practice

- Add the Google Maps URL while creating the branch so customers have the correct pickup direction immediately.
- Keep inactive branches in the list only when you still need the history.
- Recheck the assigned team list after role changes or staffing changes.
