# ClickCollect portal guide

This documentation set is focused on the operator portal side of ClickCollect.coffee. It follows the same top navigation order exposed in the live portal so owners, admins, and managers can move from the docs into the product without translation work.

## Before you start

- Use a portal account with the correct role for the section you want to edit.
- Expect some actions to trigger billing prompts when you add products, team members, locations, or domains beyond the included plan capacity.
- Branding, Billing, Locations, Team, Domains, and Settings are owner/admin areas.
- Shop Manager access changes by role: staff can work orders, managers can work orders plus menu setup, owners/admins can use the full portal.

## Portal navigation order

| Portal area | What it controls |
| --- | --- |
| Branding | Storefront theme, hero copy, layout, fonts, and colors |
| Billing | Card setup, monthly total, usage breakdown, invoices, and statements |
| Locations | Pickup branches, default branch, WhatsApp, Google Maps, and assigned team members |
| Team | Role-based access and location assignment |
| Shop Manager | Orders, categories, extras, drinks, and operational settings |
| Domains | Customer-facing custom domains and storefront provisioning |
| Settings | Company profile, payment gateways, VAT, and storefront policies |
